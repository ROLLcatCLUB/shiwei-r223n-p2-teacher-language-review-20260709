## 一、阅读说明

P2 不改 R223N 的课堂事件链，也不删学生反应、教师应对、投屏示范、学习单和评价证据。它只做一件事：把 P1 中偏系统化的标签和模板句，转写成教师更容易读的备课语言。完整结构仍保留在 review ledger，默认稿尽量像老师会拿来读、改、上课的稿子。

## 二、课时定位

《有趣的纸印》围绕“印痕是版画的独特语言”展开。学生先认识纸材肌理，再通过改造、试印、印法比较和作品展评，逐步理解纸材、印法和印痕效果之间的关系。本稿仍保留课堂节奏，但不把后端字段直接铺给老师。

## 三、教学过程

### （一）任务入场：走进版画车间坊

**这一环节要解决什么。** 把课堂从普通纸手工拉回版画语言，明确本课要寻找纸材、印法和印痕之间的关系。教师要抓住的是：不要把版画车间坊说成普通作品展，先把学生眼睛带到“印痕”上。

上课时可以这样展开。教师先把任务说清：“今天我们不是只做一张纸作品，而是进入一个小小的版画车间。请你带着一个问题开始：纸材和印法碰在一起，会留下什么样的印痕惊喜？”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能说出纸张有厚薄、软硬、粗糙、光滑等差异；能联想到拓印、印章、压印等生活经验。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“这张纸如果不画，只印，会留下什么样的痕迹？”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“比一比”，但它不是额外活动，只是帮助学生区分绘画线条与纸材印痕

需要投屏或示范时，内容保持很小：版画车间坊任务语 + 画出来/印出来对比图 + 记录句式。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：只关注作品好不好看，忘记记录纸材和印法；把版画车间坊理解成普通展板布置。教师可以这样接：如果学生仍按手工装饰理解任务，教师先展示一张材料痕迹明显的试印纸，追问“这里最有意思的是图案，还是纸留下的纹理？”再把任务缩小成先找一种痕迹。如果还需要支架，再用这一招：用一张普通绘画作品和一张拓印痕迹图作对比，圈出画线与印痕的不同。

**留下什么，再往哪走。** 本环节至少留下：能说出一个印痕观察点。学习单可以轻量记录：“我发现印痕和画线不同的地方是……”有了这条证据，就可以继续往下走：先知道为什么要看印痕，再去找印痕可能从哪里来。

**大屏、学习单和评价怎么跟上。** 大屏只放版画车间坊任务语 + 画出来/印出来对比图 + 记录句式；学习单对应“入场记录”；评价看学生是否能做到：理解印痕是版画的独特语言；明确单元任务并能进入纸材、印法、印痕的观察。

### （二）摸一摸：纸材肌理观察

**这一环节要解决什么。** 让学生用手摸、眼看、侧光观察等方式，把纸材差异转化为可以预测印痕的依据。教师要抓住的是：观察不是热身，必须留下“我预测”的理由，后面才能验证。

上课时可以这样展开。教师先把任务说清：“每组选择三种纸，不急着印。先摸一摸、看一看、斜着照一照，猜一猜哪张纸最可能留下明显肌理。”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能发现纸张厚薄、软硬、粗糙、光滑、凹凸纹理；能猜测粗糙或有纹理的纸可能更容易留下痕迹。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“你摸到的是滑，还是有一点凹凸？”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“材料盲盒”，但它不是额外活动，只是帮助学生用触觉和肌理词汇认识材料差异

需要投屏或示范时，内容保持很小：纸材观察词库 + 侧光肌理照片 + 一句预测格式。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：只按颜色或大小分类；摸完没有语言描述，只说“这个好”。教师可以这样接：如果学生只按颜色分类，教师拿两张同色但肌理不同的纸让学生闭眼触摸，追问“看不见颜色以后，你还能怎么分？”如果还需要支架，再用这一招：提供词库：粗糙、细密、凸起、凹下、吸水、光滑、柔软、硬挺。

**留下什么，再往哪走。** 本环节至少留下：每组至少一条带理由的纸材预测。学习单可以轻量记录：“我选择的纸材是……我预测它会留下……印痕，因为……”有了这条证据，就可以继续往下走：学生先知道原有肌理，再尝试制造新的肌理。

**大屏、学习单和评价怎么跟上。** 大屏只放纸材观察词库 + 侧光肌理照片 + 一句预测格式；学习单对应“纸材预测”；评价看学生是否能做到：了解各种纸材及它的纹理；能主动参与纸材收集、分类和特性分析。

### （三）变一变：制造纸的新肌理

**这一环节要解决什么。** 让学生理解纸材不是被动材料，可以通过改造制造新的可印肌理。教师要抓住的是：限制方法数量，宁可小而清楚，不要复杂但印不出来。

上课时可以这样展开。教师先把任务说清：“请你只选一种方法改造纸：刻、剪、撕、揉、卷、折或拼贴。改造后先不追求像什么，而要看看它会不会留下新的肌理。”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能尝试一种纸材改造方法；能发现揉皱、折线、撕边、拼贴厚度会影响印痕。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“你改造出来的肌理在哪里？请用手指给同伴看。”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“技法拆解”，但它不是额外活动，只是帮助学生把纸材改造拆成可操作的小动作

需要投屏或示范时，内容保持很小：刻/剪/撕/拼贴/揉/卷/折方法图 + 小块试验提醒。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：剪成复杂造型却没有可印纹理；拼贴太厚导致试印时压不平。教师可以这样接：如果学生陷入复杂手工，教师暂停 30 秒，让全班看一个简单但肌理清楚的小样，强调今天先要能印清楚，再考虑造型丰富。如果还需要支架，再用这一招：把方法限制为“一组先选一种”。

**留下什么，再往哪走。** 本环节至少留下：每组保留一个改造小样。学习单可以轻量记录：“我用……方法让纸出现了……肌理。”有了这条证据，就可以继续往下走：只有试印后，学生才能知道肌理是否真的成为印痕。

**大屏、学习单和评价怎么跟上。** 大屏只放刻/剪/撕/拼贴/揉/卷/折方法图 + 小块试验提醒；学习单对应“改造方法”；评价看学生是否能做到：能用刻、揉、卷等方法在纸上制造肌理；能将纸材改造作为版画表现语言的来源。

### （四）试一试：第一次转印

**这一环节要解决什么。** 把纸材预测转化为可观察的印痕证据，让学生建立试印、比较、记录的基本习惯。教师要抓住的是：第一次试印不是成品，关键是让学生留下可比较、可调整的证据。

上课时可以这样展开。教师先把任务说清：“每组先只做一次小试印。印完不要马上评价好不好看，先找：哪里清楚？哪里模糊？这和纸材、用力或颜料有什么关系？”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能发现清楚、模糊、断裂、灰层、重叠等印痕差异；能把试印结果和前面的纸材预测联系起来。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“你圈出的地方为什么最清楚？”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“圈一圈”，但它不是额外活动，只是帮助学生把评价落到具体可见印痕

需要投屏或示范时，内容保持很小：少量上色、均匀压印、慢揭纸三步图 + 清楚/糊印对比。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：颜料太多或太少，导致印痕糊成一片或印不出来；用力不均，无法判断是纸材问题还是操作问题。教师可以这样接：如果全班大面积糊印，教师立即暂停，拿一块小版现场示范“颜料少一点、压力均一点、揭纸慢一点”，再允许每组补一次小试印。如果还需要支架，再用这一招：示范少量上色、均匀压印和慢慢揭纸。

**留下什么，再往哪走。** 本环节至少留下：每组至少一张有圈画和一句调整计划的试印记录。学习单可以轻量记录：“第一次试印：最清楚的地方是……我想调整……”有了这条证据，就可以继续往下走：学生知道印痕会变，下一步验证不同印法如何带来不同变化。

**大屏、学习单和评价怎么跟上。** 大屏只放少量上色、均匀压印、慢揭纸三步图 + 清楚/糊印对比；学习单对应“试印记录”；评价看学生是否能做到：能用印痕来表达自己的想法；能在探索中随时记录并说明印痕变化。

### （五）比一比：干印、湿印与油印

**这一环节要解决什么。** 让学生在比较中理解印法不是名称记忆，而是会改变印痕清晰度、灰层和画面效果的选择。教师要抓住的是：不要急着讲全套术语，先让学生看见同一块版在不同印法下的差异。

上课时可以这样展开。教师先把任务说清：“同一块小版，用两种不同方法试印。请比较：哪一种边缘更清楚？哪一种灰层更丰富？哪一种更适合你想要的画面？”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能比较不同印法造成的清晰、模糊、灰层、浓淡变化；能发现同一纸材换印法后印痕不同。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“这两次试印最大的不同在哪里？请指给大家看。”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“比一比”，但它不是额外活动，只是帮助学生把技法术语转化为可见效果比较

需要投屏或示范时，内容保持很小：同版不同印法对照图 + 清晰边缘/灰层/肌理保留观察角度。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：只记住名称，无法描述差异；追求颜色浓却忽视印痕层次。教师可以这样接：如果比较变成术语背诵，教师遮住术语名称，只展示两张试印效果，让学生先说差异，再把差异对应到印法名称。如果还需要支架，再用这一招：把比较控制在两种印法，不一次展开所有术语。

**留下什么，再往哪走。** 本环节至少留下：每组至少一组同版不同印法对照。学习单可以轻量记录：“我比较了……和……，我选择……，因为它让印痕……”有了这条证据，就可以继续往下走：学生带着明确纸材、印法和效果理由进入作品。

**大屏、学习单和评价怎么跟上。** 大屏只放同版不同印法对照图 + 清晰边缘/灰层/肌理保留观察角度；学习单对应“印法比较”；评价看学生是否能做到：能掌握干印、水印、油印技法；能注意拓印时轻重缓急，让画面形成黑、灰层面。

### （六）做一做：用印痕完成作品

**这一环节要解决什么。** 引导学生把前面的纸材、肌理、印法选择转化为有画面意识的作品，而不是试印纸堆叠。教师要抓住的是：作品完成不是越满越好，要保留一个能说清来源和效果的关键印痕。

上课时可以这样展开。教师先把任务说清：“现在开始完成一幅小作品。你至少要保留一个自己满意的印痕，并说明它来自哪种纸材或印法。”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能选择一到两种纸材或印法进入作品；能保留满意印痕并围绕它组织画面。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“这幅作品里最想让大家看到的印痕是哪一处？”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“中途看一看”，但它不是额外活动，只是帮助学生把试验发现转化为画面取舍

需要投屏或示范时，内容保持很小：一主一辅画面提示 + 印痕过多/重点清楚作品对比 + 清理提示。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：把所有试过的材料都印上去，画面没有重点；为了完整图案而覆盖掉最有特色的印痕。教师可以这样接：如果画面杂乱，教师让学生用手遮住部分区域，选择最想保留的一处印痕，再围绕它减少或调整其他痕迹。如果还需要支架，再用这一招：给出一主一辅的画面建议：一个主要印痕，一个辅助印痕。

**留下什么，再往哪走。** 本环节至少留下：作品中至少有一处可说明来源的印痕。学习单可以轻量记录：“我最满意的印痕是……它来自……我保留它是因为……”有了这条证据，就可以继续往下走：作品完成后要用印痕语言解释，而不是只展示结果。

**大屏、学习单和评价怎么跟上。** 大屏只放一主一辅画面提示 + 印痕过多/重点清楚作品对比 + 清理提示；学习单对应“作品取舍”；评价看学生是否能做到：能用自己满意的印感创作新的印痕之美；能养成好习惯，学会清理油墨。

### （七）说一说：印痕展评与车间收束

**这一环节要解决什么。** 把展示从“谁的作品漂亮”转为纸材、印法、印痕效果和记录证据的综合说明。教师要抓住的是：展示时必须带试印证据，避免作品评价滑成“谁更漂亮”。

上课时可以这样展开。教师先把任务说清：“每组选择一件作品和一张试印记录一起展示。介绍时必须说清：用了什么纸材，试了什么印法，留下了什么印痕。”接着让学生直接进入观察或尝试，不急着评价作品效果。学生大多会先出现这样的反应：能把作品和试印记录一起展示；能用纸材、印法、印痕、肌理、灰层等词说明作品。这时教师顺着学生已经说出的内容往前推，而不是另起一个讲解。

    如果学生说得比较泛，教师可以追一句：“你最满意的印痕来自哪次试印？”如果还说不清，就把观察对象放大、圈出来，或让学生换一种方式再看一遍。这里可以安排一次“作品画廊”，但它不是额外活动，只是帮助学生让展评从喜好投票转为有依据的印痕说明

需要投屏或示范时，内容保持很小：展示句式 + 纸材/印法/印痕/画面四类评价词 + 证据归档提醒。不要一次把所有材料、技法和评价都放上去，只服务这一环节正在发生的观察或操作。

**如果学生卡住。** 最常见的情况是：只说作品漂亮或颜色好；忘记带上试印证据。教师可以这样接：如果展示变成投票，教师暂停投票，改为每组找一处“最能说明纸材特点的印痕”，先说证据再贴星。如果还需要支架，再用这一招：提供展示句式：我用了……纸，采用……印法，印出了……效果。

**留下什么，再往哪走。** 本环节至少留下：每组至少一份作品与试印证据配套归档。学习单可以轻量记录：“我用了……纸，采用……印法，印出了……效果；同伴建议我……”有了这条证据，就可以继续往下走：学生带着纸材、印法、印痕证据继续深化版画表现。

**大屏、学习单和评价怎么跟上。** 大屏只放展示句式 + 纸材/印法/印痕/画面四类评价词 + 证据归档提醒；学习单对应“展评归档”；评价看学生是否能做到：能欣赏作品，积极参与小组汇报讨论并与他人分享；能用印痕表达想法，并说明纸材与印法的效果。
